# 42sh

### Utilisation:
Tetris dans shell :
```
$ make re
$ ./tetris [options] [-help]
```

Tetris graphique :
```
$ cd bonus/
$ make re
$ ./tetris
```

### Informations sur le projet:

Reproduction du jeux tetris dans un terminal.

Bonus : Le jeux reproduit avec la librairie graphique CSFML et des bonus en jeu.

Plus d'informations dans tetris.pdf.
